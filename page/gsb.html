Présentation

L'application GSB est une application web permettant au . Celle-ci se connecte à une API hébergée par un serveur distant sur lequel une application java expose le service d'API REST.

Cette API REST est aussi utilisé pour envoyer des demandes d'information ou des questions sur le serveur hôte via un formulaire de contact.

L'application mobile comporte également une fonctionnalité d'ajout de formations en favori qui est enregistré sur le mobile via une base de donnée interne.

Planifier les activités

Les activités ont été planifiées grâce à l’utilisation d’un outil collaboratif en ligne dénommé Trello. Celui-ci permet de créer des tâches représentées par des « cartes » qui possèdent un titre, dans lequel on trouve le libellé de la tâche et son résumé. Les libellés doivent être utilisés dans les titres des commits sur git pour référencer les tâches.
